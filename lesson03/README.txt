Строки
----------

String в Java - это Объект, представляющий последовательность символов.
Все литералы String, например, «Привет», реализованы как объекты String.

В Java строковые объекты неизменяемы. После создания строкового объекта
его данные или состояние нельзя изменить, но создается новый строковый объект.

Класс StringBuffer используется для создания изменяемой строки.
Класс StringBuffer такой же, как класс String, за исключением
того, что он изменяемый.

Класс StringBuffer является потокобезопасным, т.е. несколько потоков
не могут получить к нему доступ одновременно. Так что это безопасно и
ведет к порядку.

Класс Java StringBuilder используется для создания изменяемой строки.
Класс Java StringBuilder аналогичен классу StringBuffer, за исключением
того, что он не синхронизирован. Он доступен с JDK 1.5.

StringBuilder не синхронизирован, т.е. не потокобезопасен. Это означает,
что два потока могут одновременно вызывать методы StringBuilder.
